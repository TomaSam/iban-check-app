# International Bank Account Number (IBAN) validation

## Features

* User can input IBAN and check if it is valid 
* User can input IBAN and find out Bank name of entered IBAN
* User can enter the path of CSV file with IBAN items and export CSV file with validated IBAN numbers
* User can enter the path of CSV file with IBAN items and export CSV file with Bank names of IBAN numbers

## Installation

In the project directory iban, run:

```mvn
mvn clean install
```

```mvn
mvn spring-boot:run
```

In the project directory iban-frontend, run:

```npm
npm install
```

```npm
npm start
```
